# Building Desktop and Mobile apps with Vite

## Electron

- Experimental support via [Electron Forge](https://www.electronforge.io/)
  - https://www.electronforge.io/config/plugins/vite
- Also https://github.com/alex8088/electron-vite exists. TODO: Why one or the other?!

## Tauri

- No Vite plugin, slight config changes necessary
  - https://tauri.app/start/frontend/vite/

## NativeScript

- Works with Vite and ESM now
  - https://blog.nativescript.org/nativescript-9-announcement/

## React Native with Vite?

- No official integration
- Community integration for React Native Web application exists
  - https://github.com/dannyhw/vite-plugin-rnw

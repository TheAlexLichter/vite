# Framework Integration

## Frameworks

- Official support via Vite

  - React
  - Vue

- Maintained by the respective framework teams
  - Svelte
  - Preact
  - Angular (dev-server only)
  - Solid
  - Qwik
  - [Marko](https://github.com/marko-js/vite)
  - Ember

## Meta Frameworks

Hint to full-stack integration guide

- Meta frameworks that are "just Vite plugins":

  - SvelteKit
  - Tanstack Start
  - SolidStart
  - Analog
  - Redwood SDK

- Meta frameworks that are based on Vite:
  - Astro
  - Nuxt.js
  - Waku
  - VitePress

### Unsupported Meta Frameworks

- Next.js
- No Vite integration -> TL;DR why not
